🎬 Matrix Digital Rain Effect

\n

Production-ready HTML5 Canvas component with A/B testing support and GTM event tracking.

\n

Features

\n

✅ 60fps Canvas rendering with auto-pause on hidden tab
✅ A/B test variants (Oranje vs Green) with localStorage persistence
✅ GTM event tracking for analytics integration
✅ Responsive design for mobile & desktop
✅ Accessibility: prefers-reduced-motion support
✅ No dependencies, pure vanilla JavaScript

\n

Integration

\n

Add to your HTML <head> section:

\n 
<link rel=\"stylesheet\" href=\"https://[your-netlify-url]/matrix-rain.css\">\n<script src=\"https://[your-netlify-url]/matrix-rain.js\"></script>\n<script>\n  initMatrixRain({ variant: 'auto' });\n</script>
\n

GTM Events

\n

Events fired to window.dataLayer:

\n 
matrix_variant_loaded    // Component initialized\nanalysis_completed       // User completed analysis\nanalysis_abandoned       // User abandoned analysis\ncta_clicked              // CTA button clicked
\n

All events include matrix_variant dimension for A/B test attribution.

\n

Testing

\n

Use the control panel (top-right) to switch variants, adjust animation speed, run FPS tests, and view system information.

\n

Deployment

\n

1. Push files to GitHub
2. Connect repo to Netlify → auto-deploy
3. Use custom domain: matrix.profielscore.nl
4. Link from profielscore.nl and test variants
5. Monitor GA4 events for 2 weeks or 200 sessions
